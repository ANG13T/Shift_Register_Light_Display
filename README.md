# 74HC595 Shift Register Light Display

This projects utilizes a 74HC595 shift register to make a display with LEDS!

## Materials
- 20 - 30 jumper wires
- 74HC595 shift register
- 5 x 220ohm resistors
- 5 LEDs
- Arduino Uno
- Breadboard
